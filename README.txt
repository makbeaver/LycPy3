Суть проекта: 
Сайт на Flask с использованием изученных технологий (Библиотека SQLAlchemy, Работа с HTML шаблонами, Подключение Bootstrap и стилей CSS)
Сайт представляет из себя онлайн-магазин, с авторизацией и регистрацией, добавлением товаров в корзину(удаление товаров из корзины), подсчёт общей цены.
Также реализован аккаунт администратора магазина, где можно добавлять новые товары или удалять старые.
Имеется возможность смены пароля на сайте.

Структура проекта:
9 шаблонов HTML
1 скрипт JavaScript
1 база данных с 2 таблицами(товары и пользователи)
1 файл со стилями CSS
Основной файл с программой
